For each game:
1. MARKET DISCREPANCY
   - Identify best available price (Side, Total, ML).
   - Note bookmaker outliers.
   - Convert decimal odds to implied probability (1 / odds).
2. TEAM EVALUATION (Score each category)
   Assign ratings:
   - Offensive Form (1–10)
   - Defensive Stability (1–10)
   - Injury Impact (-3 to 0)
   - Fatigue / Schedule Score (-3 to +1)

   - Back-to-back: -2
   - 3 games in 4 nights: -1.5
   - Cross-country travel: -1
   - 2+ days rest: +1
3. SITUATIONAL EDGE
   - Evaluate matchup advantages (paint defense, 3PT variance, pace).
   - Explicitly account for key injuries (OUT / DOUBTFUL / QUESTIONABLE).
   - Mention how absence affects offense, defense, rotation, or pace.
4. IMPLIED EDGE CHECK
   - Convert odds to implied probability.
   - Assign a realistic estimated win probability.
   - Only recommend if: Estimated Probability ≥ Implied Probability + 3%.
5. MARKET STRENGTH SIGNAL
   - If 3+ books align within 0.03 odds → strong consensus.
   - If 1 book is an outlier by >0.05 → potential value.
If no plays qualify → output: "No qualified plays today."
